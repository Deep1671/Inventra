## **🚀 QUICK START GUIDE**

### **Step 1: Install Ollama** (5 minutes)
```
1. Go to: https://ollama.ai
2. Download and install Ollama
3. Test: ollama --version
```

### **Step 2: Start Services** (4 terminals)
```bash
# Terminal 1: Start Ollama service
ollama serve

# Terminal 2: Download model (one-time)
ollama pull mistral

# Terminal 3: Start backend
cd smart_Inventory_management/backend
npm start

# Terminal 4: Start frontend
cd smart_Inventory_management/frontend
npm run dev
```

### **Step 3: Access & Test**
```
1. Open: http://localhost:5173/insights
2. Select language: English or Hindi
3. Click a button: Inventory, Sales, Supplier, or Actionable
4. Get instant AI insights!
```

## **❓ COMMON QUESTIONS**

**Q: Do I need an API key?**
A: No! Everything runs locally. No API keys or subscriptions needed.

**Q: Is my data safe?**
A: Yes! 100% local processing. No data leaves your server.

**Q: Can I use different models?**
A: Yes! Change `OLLAMA_MODEL` in `.env` and restart.

**Q: What if I run out of RAM?**
A: Use smaller model (orca-mini) or add more RAM.

**Q: Can I add more languages?**
A: Yes! Modify language templates in the service file.

**Q: Will this work offline?**
A: Yes! Once Ollama is installed, it works completely offline.
